# Charts

This section contains the description of charts in a dashboard. It is an array of charts. Here's an example:

```json
{
  "charts": [
    { "type": "line", "data": "table",
      "position": {
        "columns": 1,
        "rows": 1,
        "x": 0,
        "y": 0
      },
      "axes": {
        "x": {
          "label": "name",
          "dataColumn": "name"
        },
        "y": {
          "label": "count",
          "dataColumns": [
            {"name": "uv", "color": "#8884d8"},
            {"name": "pv", "color": "#82ca9d"}
          ]
        }
      }
    }
  ]
}
```

Each chart contains the below fields.

- type: type of the chart
- title: title of the chart
- data: name of the dataset. It references the name field in the data section.
- position: position of the chart on dashboard. See #Layout page for how to use layout.
  * columns: number of columns the chart occupies
  * rows: number of rows the chart occupies
  * x: horizontal position of the chart on the dashboard
  * y: vertical position of the chart on the dashboard
- axes: axes for the chart
  * x: horizontal axis
  * y: vertical axis

### Chart Types

We support the following chart types.

- Line
- Bar
- Pie
- Scatter
